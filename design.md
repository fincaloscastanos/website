# CORPORATE IDENTITY & CORPORATE DESIGN

*Finca Los Castaños · Valle de Agaete · Gran Canaria*

## 1. MARKENIDENTITÄT

Finca Los Castaños ist ein nachhaltiges Landgut im Valle de Agaete, im Nordwesten Gran Canarias. Die Marke steht für authentischen Kaffeeanbau, Naturnähe, Gastfreundschaft und nachhaltige Lebensweise. Das CI/CD-System spiegelt Wärme, Natur und Handwerkskunst wider.

| Markenwert | Beschreibung |
|---|---|
| Name | Finca Los Castaños |
| Untertitel | Valle de Agaete |
| Webseite | fincaloscastanos.com |
| Sprachen | Deutsch · Español · English |
| Positionierung | Nachhaltiger Kaffeeanbau, Agrotourismus, Eventlocation |
| Zielgruppe | Naturliebhaber, Kaffeeinteressierte, Event-Kunden, Urlauber |

## 3. TYPOGRAFIE

Es werden zwei Schriftfamilien kombiniert: eine serifenlose Sans-Serif für klare Strukturen und eine elegante Kursivschrift für emotionale Akzente.

| Schriftart | Font | Stil | Größe (ca.) | Verwendung |
|---|---|---|---|---|
| Serifenlose (Sans-Serif) | **Raleway** | Light / Regular | 11–13 px | Haupttitel (VERSALIEN), Navigation, Fließtext |
| Kursiv-Script | **Oooh Baby** | Italic Regular | 14–18 px | Untertitel, emotionale Akzente, Slogans |
| Sans-Serif Bold | **Raleway** | Bold | 12–14 px | Buttons, Hervorhebungen im Text |
| Handschrift (Schild) | – | Irregular | Dekoration | Physisches Schild auf der Finca |

| Element | Font | Beschreibung |
|---|---|---|
| Hauptüberschriften | Raleway | VERSALIEN, sehr leicht, serifenlos, dunkelgrau (#2C2C2C) |
| Untertitel | Oooh Baby | Kursive Scriptschrift, dunkelgrün (#2D6A4F) oder schwarz |
| Fließtext | Raleway | Serifenlos, zentriert auf Mobile, grau (#444444), ca. 16 px |
| Links / Highlights | Raleway | Salbeigrün (#8BA888) oder Dunkelgrün (#2D6A4F), normal/bold |
| Zitat-Text | Raleway | Große VERSALIEN, sehr weit gesperrt, dunkelbraun (#3D1E0E) |

## 5. LAYOUT & DESIGN-PRINZIPIEN

| Prinzip | Beschreibung |
|---|---|
| Bildsprache | Hochwertige Fotografie: Nahaufnahmen von Kaffeebohnen, Maschinen, Pflanzen, Team und Landschaft. Natürlich, warm, authentisch. |
| Hintergründe | Weiß, Hellgrau (#F0EFED), Hellbeige (#F5E6CA) und Warmgold (#D4A84B) als Hintergrundflächen für Sektionen. |
| Bildformat | Rechteckig, vollbreit auf Mobile. Bildergalerie als 2-spaltige Grid. |
| Textausrichtung | Zentriert (Mobile-first), großzügige Abstände. |
| Buttons / CTAs | Abgerundete Rechtecke in Salbeigrün (#8BA888), weiße Schrift. Auf goldenem Hintergrund: outline-style (transparent mit weißem Rahmen). |
| Dekor. Elemente | Goldene Pinsel-/Watercolor-Flächen als Overlay-Akzent zwischen Sektionen. |
| Navigation | Sticky Header mit Logo links, Flaggen und Hamburger-Menü rechts. Menü öffnet als Slide-in von rechts. |
| Menüstruktur | Erleben · Kaffeetour buchen · Veranstaltungszelt buchen · Entspannen · Entdecken · Kontakt |
